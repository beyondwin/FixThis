# FixThis v0.2.2

FixThis v0.2.2 is a patch release for Maven Central publication automation on
the public agent-first install path.

## What changed

- Maven Central publishing now creates a signed Maven repository bundle and
  uploads it through the Central Portal Publisher API.
- The publish workflow polls the Central deployment state so a successful run
  means the deployment reached `PUBLISHED`, not just "uploaded".
- Public install docs and CLI defaults now point agents at `0.2.2`.

## Agent install

From a Jetpack Compose Android app repository:

```bash
curl -fsSL https://raw.githubusercontent.com/beyondwin/FixThis/main/scripts/install-fixthis.sh \
  | bash -s -- --version v0.2.2

fixthis install-agent --project-dir . --target all
./gradlew fixthisSetup
fixthis doctor --project-dir . --json
```

Restart Claude Code or Codex after MCP config is written, then use
`fixthis_open_feedback_console`.
