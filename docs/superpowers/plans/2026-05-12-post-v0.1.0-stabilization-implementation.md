# Post-v0.1.0 Stabilization Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** `v0.1.0..HEAD` 리뷰에서 발견한 production event log, compaction, fingerprint, stale/recovery, CI guard 결함을 실제 런타임 경로까지 닫는다.

**Architecture:** 먼저 deterministic build/test guard를 세운 뒤, session persistence를 event-log-backed production store로 연결한다. Compaction은 checkpoint 기반 replay로 안전하게 만들고, preview save는 server-side live recapture fingerprint 비교를 통과해야만 session mutation을 수행한다. Console pending recovery는 item 배열 대신 frozen preview context envelope을 저장한다.

**Tech Stack:** Kotlin/JVM 21, kotlinx.serialization, Java `File`/NIO atomic move, vanilla console JS concatenated by `scripts/build-console-assets.mjs`, Node built-in test runner, Gradle/Android/JUnit.

**Related spec:** [`../specs/2026-05-12-post-v0.1.0-stabilization-detailed-spec.md`](../specs/2026-05-12-post-v0.1.0-stabilization-detailed-spec.md)

In `pendingPersistence.js`, keep backward compatibility:

```js
function persistPendingState(sessionId, value) {
  if (!sessionId || typeof localStorage === 'undefined') return;
  const envelope = {
    schemaVersion: 1,
    sessionId,
    previewId: value?.previewId ?? null,
    screen: value?.screen ?? null,
    screenshotUrl: value?.screenshotUrl ?? null,
    frozenAtEpochMillis: value?.frozenAtEpochMillis ?? Date.now(),
    items: Array.isArray(value?.items) ? value.items : [],
  };
  localStorage.setItem(pendingKey(sessionId), JSON.stringify(envelope));
}

function restorePendingState(sessionId) {
  const raw = localStorage.getItem(pendingKey(sessionId));
  if (!raw) return null;
  const parsed = JSON.parse(raw);
  if (Array.isArray(parsed)) {
    return { schemaVersion: 0, sessionId, previewId: null, screen: null, screenshotUrl: null, frozenAtEpochMillis: null, items: parsed };
  }
  return parsed && Array.isArray(parsed.items) ? parsed : null;
}
```
